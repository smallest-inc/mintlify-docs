## Model Overview

| Model ID              | Description                                                                 | Languages Supported |
|-----------------------|-----------------------------------------------------------------------------|---------------------|
| **lightning**         | Fastest model with an RTF of 0.01, generating 10 seconds of audio in 100 ms. | `English` <br/> `Hindi`           |
| **lightning-large**   | More emotional depth and expressiveness, supports voice cloning, latency under 300 ms. | `English`  <br/> `Hindi` <br/> `Tamil` <br/> `German` <br/> `French` <br/> `Polish`            |
| **lightning-v2** | Supports 16 languages, currently in beta.                                 | `English` <br/> `Hindi` <br/> `Tamil` <br/> `Kannada` <br/> `Gujarati` <br/> `Bengali` <br/> `Marathi` <br/> `German` <br/> `French` <br/> `Spanish` <br/> `Italian` <br/> `Polish` <br/> `Dutch` <br/> `Russian` <br/> `Arabic` <br/> `Hebrew`      |


<Note>
Note: The API uses [ISO 639-1 language codes -  Set 1](https://en.wikipedia.org/wiki/List_of_ISO_639_language_codes) (2-letter codes) to specify supported languages.
</Note>
