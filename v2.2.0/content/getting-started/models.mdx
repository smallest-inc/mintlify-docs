## Model Overview

| Model ID              | Description                                                                 | Languages Supported |
|-----------------------|-----------------------------------------------------------------------------|---------------------|
| **lightning**         | Fastest model with an RTF of 0.01, generating 10 seconds of audio in 100 ms. | English, Hindi             |
| **lightning-large**   | More emotional depth and expressiveness, supports voice cloning, latency under 300 ms. | English, Hindi, Tamil, German, French, Polish             |
| **lightning-multilingual** | Supports 30 languages, currently in beta.                                 | 30 languages        |


<Note>
Note: The API uses [ISO 639-1 language codes](https://en.wikipedia.org/wiki/List_of_ISO_639_language_codes) (2-letter codes) to specify supported languages.
</Note>
